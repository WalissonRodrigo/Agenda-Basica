# Agenda Básica

Aplicativo de agenda simples desenvolvido em **C# Windows Forms**.

## Sobre

Sistema desktop para cadastro de contatos, com campos para nome, telefone, e-mail e endereço. Projeto de estudo de C# e .NET Framework.

## Tecnologias

- C#
- Windows Forms
- .NET Framework
- MySQL

## Configuração

1. Abra a solução no Visual Studio.
2. Configure a string de conexão no `App.config`.
3. Compile e execute o projeto.
